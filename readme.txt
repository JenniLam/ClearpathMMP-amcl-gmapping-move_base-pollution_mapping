Pollution Mapping

The cear_jenni package contains all nodes, urdf files, and maps necessary for slam_gmapping, amcl, move_base, and pollution mapping.
Extract this .zip file in the src folder of your catkin workspace. In the catkin workspace folder that contains the build, devel, and src folders, invoke catkin_make.

The PollutionMapping.pdf file contains basic instructions for all other aspects of the package.
